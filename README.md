## 一句话概述

```
最终概率 = 泊松分布(
              环境修正 × 
              行为金融修正 × 
              定性情报修正 × 
              多源强度合成( Elo 55% + 俱乐部质量 25% + 近期状态 20% )
          )
```

输入两队的竞彩赔率，输出胜/平/负的数学概率、Edge Score（期望收益）、以及建议投注金额。

**核心哲学：你不是在跟精算师对赌——你是在跟昨天看完揭幕战头脑发热的散户对赌。**

## 数学模型全流程

### 第一步：True Elo —— 联合会偏差修正

原始 Elo 积分来自 `eloratings.net`，但它有一个致命缺陷：**Elo 是联合会内部的零和博弈**。亚洲球队在 AFC 内部刷弱队涨分，跟欧洲球队在 UEFA 55 队血海里拿分，质量完全不同。

| 联合会 | 修正值 | 原因 |
|--------|--------|------|
| UEFA（欧洲） | **±0** | 55 队深度竞争，基准线 |
| CONMEBOL（南美） | **−15** | 10 队双循环无鱼腩，分数被结构性压低 |
| CONCACAF（中北美） | **−40** | 前 6 后 29 实力断崖 |
| AFC（亚洲） | **−25** | 日韩伊澳沙之外骤降 |
| CAF（非洲） | **−20** | 埃及塞内加尔外竞争偏弱 |
| OFC（大洋洲） | **−55** | 新西兰独大 |

```
True_Elo = Raw_Elo + Confed_Adjustment
```

**实例**：日本原始 Elo ≈ 1900，AFC 修正 −25 → True Elo = 1875。表面跟荷兰只差 20 分，真实差 45 分。

---

### 第二步：多源强度合成 —— 不只靠 Elo

Elo 只反映"过去的胜负"，但世界杯级别的比赛还需要考虑 **球员质量** 和 **近期状态**。

```
Final_Strength = True_Elo × 0.55 + Club_Quality × 0.25 + Recent_Form × 0.20
```

**俱乐部质量 (Club_Quality)**：

| 变量 | 含义 | 数据来源 |
|------|------|---------|
| Big5_Minutes_Pct | 全队球员在五大联赛的出场时间占比 | FBref |
| TM_Value_Percentile | 全队德转身价在 48 队中的百分位 | Transfermarkt |

```
Club_Quality = (Big5_Minutes_Pct × 0.5) + (TM_Value_Percentile × 0.5)
```

**近期状态 (Recent_Form)**：

```
Recent_Form = PPG_last_10 × Opponent_Strength_Multiplier
```

- `PPG_last_10`：近 10 场场均积分（胜=3, 平=1, 负=0），**优先取世预赛，其次洲际杯，友谊赛权重最低**
- `Opponent_Strength_Multiplier`：对手平均 Elo ÷ 全球平均 Elo（≈1750）

> ⚠️ 这就是"日本世预赛 PPG 2.60 为什么不如荷兰的 2.10"——日本对手平均 Elo ≈ 1550（中国、巴林、印尼），乘数只有 0.886。荷兰对手均 Elo ≈ 1686，乘数 0.963。表面 2.60 > 2.10，修正后 2.30 ≈ 2.02——差距从 0.50 压缩到 0.28。

### 第三步：泊松分布 —— 把强度转化为进球

足球比赛的进球数近似服从 **泊松分布（Poisson Distribution）**：

```
P(进 k 球) = (λᵏ × e⁻λ) / k!
```

其中 λ（lambda）是预期进球数，通过两队的攻击/防守强度交叉计算：

```
λ(主队) = 基准主场 xG(1.50) × 主队攻击强度 × 客队防守强度 × 主场因子(1.15)
μ(客队) = 基准客场 xG(1.10) × 客队攻击强度 × 主队防守强度
```

**攻击强度** = 近 20 场场均进球 ÷ 全球均 GF(1.40)
**防守强度** = 近 20 场场均失球 ÷ 全球均 GA(1.40)

#### Elo 交叉验证（防过拟合）

纯攻防系数有一个致命问题：**弱队在弱联合会零封弱对手，防守强度被极端低估。**

例如日本世预赛 16 场仅丢 3 球 → Def=0.214 → 荷兰 λ_raw=0.63（荒谬）。

当攻防系数推算的 λ/μ 与 Elo 经验插值偏差 **>20%** 时，自动触发加权修正：

```
λ_final = λ_攻防 × 0.6 + λ_Elo × 0.4
```

| ΔTrue_Elo | 经验 λ | 经验 μ |
|-----------|--------|--------|
| 0 | 1.35 | 1.15 |
| 50 | 1.55 | 1.05 |
| 100 | 1.75 | 0.95 |
| 200 | 2.15 | 0.75 |

### 第四步：Dixon-Coles τ 修正 —— 平局被低估的根本原因

普通泊松分布假设两队进球相互独立。但实际足球比赛中，**0-0、1-0、0-1、1-1 发生的频率远高于独立假设**——因为球队在僵局时会互相试探、保守应对。

**Dixon-Coles 模型** 用一个 τ 函数修正这四种比分：

```
ρ = −0.13（国际赛事经验值）

τ(0,0) = 1 − λμρ      ← 低比分僵局概率 ↑
τ(1,0) = 1 + μρ        ← 1-0 修正
τ(0,1) = 1 + λρ        ← 0-1 修正
τ(1,1) = 1 − ρ         ← 1-1 概率 ↑（1.13×基准）
τ(其他) = 1             ← 高比分不修正
```

**联合概率**：

```
P(比分 x:y) = τ(λ,μ,x,y) × Poisson(x|λ) × Poisson(y|μ)
```

将双方各 0-7 球的 64 种比分汇总：

```
主胜 = Σ(x>y) P(x,y)
平局 = Σ(x=y) P(x,y)
客胜 = Σ(x<y) P(x,y)
```

**这是整个模型中平局概率系统性高于市场定价的数学根源。** 市场定价依赖简单的泊松或逻辑回归，不会对 0-0 和 1-1 做 τ 修正——尤其是当双方进攻核心同时缺阵时，低比分僵局概率飙升，市场的反应是滞后的。

### 第五步：环境修正 —— 乘性叠加

所有环境因子以 **乘法** 叠加到 λ/μ 上：

| 因素 | 阈值 | 修正 |
|------|------|------|
| 海拔 | ≤500m / 500-1500m / 1500-2500m / >2500m | 0% / −2~−3% / −5~−8.5% / −8~−13% |
| 气温 | 5-25°C / 25-30°C / 30-35°C / >35°C | 0% / −1% / −3% / −5% |
| 湿度 | <70% / 70-85% / >85% | 0% / −1% / −2% |
| 旅途 | <1500km / 1500-3000km / 3000-5000km / >5000km | 0% / −2% / −4% / −7% |
| 时差 | 0-2 / 3-5 / >5 时区 | 0% / −2% / −4% |
| 休整 | <3天 / 3-4天 / 5+天 | −5% / −2% / 0% |
| 伤病 | 0/1/2/3+ 核心缺阵 | 0% / −3% / −6% / −8% |

```
λ_adjusted = λ_base × (1 + 海拔 + 气温 + 湿度 + 旅途 + 休整 + 伤病)
```

### 第六步：行为金融修正 —— Alpha 的真正来源

传统数据（Elo、身价、战绩）反映的是"已经发生的事"。这一层捕捉的是"正在发生但数据还没来得及反应的事"——散户情绪、叙事驱动、伤病突发。

四个因子独立计算、线性叠加，**封顶 ±12pp**：

#### ① 市场情绪溢出（抽水膨胀法）

```
竞彩抽水率 = (1/主胜赔率 + 1/平赔率 + 1/客胜赔率 − 1) × 100%
情绪溢价 = max(0, 抽水率 − 11%竞彩基准)

| 情绪溢价 | 含义 | 热门队修正 | 冷门队修正 |
|---------|------|----------|----------|
| 0-3% | 正常波动 | 0pp | 0pp |
| 3-8% | 明显情绪 | −3pp | +3pp |
| 8-15% | 严重情绪 | −5pp | +5pp |
| >15% | 极度扭曲 | −8pp | +8pp |
```

**直觉**：如果某场比赛抽水飙到 18%（远超 11% 基准），说明散户一边倒下注，庄家通过加大抽水来对冲——热门方被"买过头"了，冷门方出现价值洼地。

#### ② 标志性胜利余温（Elo 滞后法）

一支弱队爆冷击败强队后，Elo 只更新了一次，但球队心理和外部预期的提升远不止这些分。

```
触发条件：击败 Elo 比自己高 ≥150 的对手，且在正式比赛（非友谊赛）

心理势能 = (对手Elo − 本队Elo) / 400 × 比赛性质权重 × 30
比赛性质权重：世界杯 1.0 | 洲际杯 0.8 | 预选赛 0.6 | 友谊赛 0.3

时间衰减：衰减因子 = e^(−0.02 × 距今场数)
修正值 = 心理势能 × 衰减因子 / 100 → 封顶 ±5pp
```

#### ③ 核心伤病影响力（身价占比法）

```
球员影响力 = (身价占比 × 0.5) + (国家队 xG参与率 × 0.5)
缺阵修正 = −影响力 × 0.7 × 100 → 转为 pp

| 影响力 | 分类 | 修正 |
|--------|------|------|
| >15% | 头号核心（如姆巴佩） | −4pp |
| 10-15% | 二号核心（如三笘薰） | −2pp |
| 5-10% | 主力 | −1pp |
| <5% | 角色球员 | 0 |
```

#### ④ 东道主/主场叙事压力

查世界杯历史（1994-2022）东道主首场表现偏差数据库，按东道主 Elo 档位给修正：

| 东道主 Elo | 历史真实加成 |
|-----------|------------|
| >1900（强队） | +1pp |
| 1700-1900（中游） | +2pp |
| <1700（弱队） | 不加修正 |

### 第七步：定性情报修正 —— v3.0 新增

搜索引擎能找到、但结构化数据库里没有的信息。**弥补纯数学模型与资深赌球专家之间的信息差。**

| 维度 | 搜索内容 | 量化映射 |
|------|---------|---------|
| 教练战术风格 | 压迫倾向 + 防守深度 | 双高位/一高一低/双低位 → 平局 ±0~2pp |
| 战术克制关系 | A 的强侧 vs B 的弱侧 | 单向克制 ±2pp |
| 球队内部氛围 | 队长更替、更衣室矛盾 | 肾上腺素 ±1pp |
| 裁判尺度 | 场均黄牌、红牌倾向 | 比赛流畅度 → λ ±2% |
| 赛前训练情报 | 关键球员复出/新阵型 | 伤病修正撤销/置信度下调 |

信息源分级（A=官方发布会 B=权威媒体交叉验证 C=单一来源 D=丢弃），A/B 全额计入，C 减半。

**封顶 ±3pp/维度，总计 ±5pp。** 搜索引擎能找到的情报，精算师也能找到——真正的信息壁垒不在这里。

## Edge Score：统一的决策语言

### 公式

```
Edge_Score = (P_model − P_market) × Odds
```

- `P_model`：模型最终概率
- `P_market`：竞彩去水后的市场隐含概率
- `Odds`：该方向的竞彩赔率
- **Edge_Score = 每投入 1 元，数学期望净利润（元）**

### 决策阈值（竞彩基准）

竞彩正常抽水 11%，比国际庄家（5-7%）高，所以阈值更严：

| Edge Score | 判定 | 动作 |
|-----------|------|------|
| **≥ 18%** | 🟢 正期望 | 买入。竞彩 11% 抽水 + 模型误差都打穿了 |
| **10-18%** | 🟡 边缘 | 可买可不买，取决于定性判断把握 |
| **< 10%** | 🔴 负/微利 | 不买。扣掉抽水后几乎是确定负期望 |

### 为什么强队难出 🟢

| 方向 | 赔率范围 | 需概率差 |
|------|---------|---------|
| 大热门 | 1.20-1.60 | 需 11-15pp |
| 中游 | 1.80-2.50 | 需 7-10pp |
| 冷门 | 3.00-5.50 | 需 3-6pp |

**低赔率天然压制 Edge。** 在竞彩买强队极难——抽水 11% + 赔率低双重压制。中游和冷门才是主战场。

### 仓位：1/4 分数凯利（Fractional Kelly）

```
全凯利（严禁使用）：f* = (bp − q) / b
1/4 凯利（本模型）：f* = (1/4) × (bp − q) / b
```

- b = 赔率 − 1
- p = 模型概率
- q = 1 − p

**为什么用 1/4？** 全凯利假设模型概率完美精确——实际不可能。世界杯只有 64 场，样本太小，全凯利会过度下注。

在每天只有 2 场比赛的稀疏机会集中，上调至 **1/2 Kelly**。

```
用户：荷兰vs日本 1.86 3.38 3.38 瑞典vs突尼斯 1.69 3.30 4.26
```

模型自动执行：

1. **结构化数据采集**：Elo、德转身价、近 10 场战绩、场地环境、伤病——5 步必选搜索
2. **定性情报采集**：教练战术风格、战术克制关系、球队氛围、裁判尺度——4 步必选搜索
3. **多语言搜索**：日语、瑞典语、阿拉伯语等母语媒体——非英语球队必选
4. **全链路计算**：True Elo → 多源合成 → λ/μ → 环境修正 → Dixon-Coles → 行为金融 → 定性修正 → Edge Score → Kelly
5. **输出**：完整概率表 + Edge Score + 建议仓位 + 残差警告

### 环境要求

- Claude Code（AI 编程助手）
- 网络搜索能力（模型依赖实时 WebSearch 获取 Elo、身价、伤病、战术情报）
- 无外部 API 依赖（所有计算在模型内部完成）

## 局限与免责

1. **xG 数据稀疏**：非欧洲/南美预选赛没有公开 xG，使用实际进球作代理
2. **俱乐部数据盲区**：只覆盖五大联赛，荷甲/葡超/巴甲球员被系统性低估
3. **伤病延迟**：Transfermarkt 更新可能在赛前 24 小时内变化
4. **定性情报信噪比**：教练发布会可能是烟雾弹，更衣室报道可能是流量驱动
5. **模型输出是基线**：一场足球比赛的随机性远超任何模型能捕捉的范围
6. **长期正 EV 需要大量样本**：概率优势在单场上只是"更可能"，在 100+ 场上才变成"确定赚"
7. **这不是投资建议**：竞彩有风险，量力而行
