# 世界杯胜平负概率计算器（独立版）

> 把以下内容完整复制给任何 AI（ChatGPT、DeepSeek、Kimi 等），
> 附上比赛信息和赔率，就能得到完整的预测结果。
> 也可以照着公式自己在 Excel 里算。

## 📋 使用方式

**复制整段话发给 AI：**

```
请严格按照以下公式，计算 {主队} vs {客队} 的胜平负概率。

已知信息：
- {主队}：原始Elo≈{xxx}，联合会{xxx}，近20场场均进球{x.xx}、失球{x.xx}，近10场PPG={x.xx}，对手均Elo≈{xxx}，全队身价€{x.xx}亿，5大联赛时间占比{xx}%，核心缺阵{xxx}
- {客队}：原始Elo≈{xxx}，联合会{xxx}，近20场场均进球{x.xx}、失球{x.xx}，近10场PPG={x.xx}，对手均Elo≈{xxx}，全队身价€{x.xx}亿，5大联赛时间占比{xx}%，核心缺阵{xxx}
- 竞彩赔率：胜{x.xx} / 平{x.xx} / 负{x.xx}
- 场地：{城市}，海拔{xxx}m，气温{xx}°C，湿度{xx}%

{下面粘贴"计算步骤"部分的全部内容}
```

## 计算步骤

### 步骤1：联合会偏差修正

```
True_Elo = 原始Elo + 联合会修正值

联合会修正值：
  UEFA（欧洲）      = 0
  CONMEBOL（南美）  = -15
  AFC（亚洲）       = -25
  CAF（非洲）       = -20
  CONCACAF（中北美）= -40
  OFC（大洋洲）     = -55

示例：日本原始Elo=1900，AFC=-25 → True_Elo=1875
```

### 步骤2：俱乐部质量分数

```
48队身价排名参考（2026年6月）：
  前5：法国15.2亿 英格兰13.6亿 西班牙12.2亿 葡萄牙10.1亿 德国9.47亿
  6-10：巴西9.28亿 阿根廷7.83亿 荷兰7.54亿 挪威5.90亿 比利时5.48亿
  11-15：科特迪瓦5.22亿 塞内加尔4.78亿 土耳其4.74亿 摩洛哥4.48亿 瑞典4.06亿
  16-20：克罗地亚3.87亿 乌克兰3.50亿 丹麦3.50亿 塞尔维亚3.50亿 瑞士2.80亿
  21-30：日本2.71亿 波兰2.50亿 韩国2.20亿 奥地利2.50亿 埃及2.50亿 ...
  31-40：尼日利亚3.50亿（注意排名） 伊朗1.50亿 澳大利亚1.80亿 ...
  41-48：突尼斯0.70亿 沙特0.50亿 卡塔尔0.50亿 新西兰0.30亿 ...

TM_Value_Percentile = (48 - 身价排名) / 47

Club_Quality = (Big5联赛时间占比 × 0.5) + (TM_Value_Percentile × 0.5)

归一化到Elo量纲（均值假设0.42，标准差0.24）：
Club_Quality_Elo = 1500 + (Club_Quality - 0.42) / 0.24 × 200
```

### 步骤3：近期状态

```
PPG_last_10 = (近10场胜场数×3 + 平场数×1) / 10
对手强度乘数 = 对手平均Elo / 1750
Recent_Form_原始 = PPG_last_10 × 对手强度乘数

归一化到Elo量纲（均值假设1.72，标准差0.42）：
Recent_Form_Elo = 1500 + (Recent_Form_原始 - 1.72) / 0.42 × 200
```

### 步骤4：综合强度指数

```
Final_Strength = True_Elo × 0.55 + Club_Quality_Elo × 0.25 + Recent_Form_Elo × 0.20
Δ = 主队Final_Strength - 客队Final_Strength
```

### 步骤5：攻击防守强度

```
主队攻击强度 = 主队近20场场均进球 / 1.40
主队防守强度 = 主队近20场场均失球 / 1.40
客队攻击强度 = 客队近20场场均进球 / 1.40
客队防守强度 = 客队近20场场均失球 / 1.40
```

### 步骤6：预期进球（攻防系数法）

```
λ_攻防 = 1.50 × 主队攻击强度 × 客队防守强度 × 1.15(主场因子)
μ_攻防 = 1.10 × 客队攻击强度 × 主队防守强度
```

### 步骤7：Elo经验插值交叉验证

```
根据Δ（步骤4的结果）查表：

| ΔTrue_Elo | λ_Elo | μ_Elo |
|-----------|-------|-------|
| 0         | 1.35  | 1.15  |
| 50        | 1.55  | 1.05  |
| 100       | 1.75  | 0.95  |
| 200       | 2.15  | 0.75  |

λ_Elo = 1.35 + (Δ/50) × 0.20（线性插值）
μ_Elo = 1.15 - (Δ/50) × 0.10

计算偏差：
  λ_偏差 = |λ_攻防 - λ_Elo| / λ_Elo
  μ_偏差 = |μ_攻防 - μ_Elo| / μ_Elo

如果λ_偏差 > 0.20（20%）：λ = λ_攻防×0.6 + λ_Elo×0.4
否则：λ = λ_攻防

如果μ_偏差 > 0.20：μ = μ_攻防×0.6 + μ_Elo×0.4
否则：μ = μ_攻防
```

### 步骤8：环境修正

```
海拔修正：
  ≤500m:      主队+0%,  客队+0%
  500-1500m:  主队-2%,  客队-3%
  1500-2500m: 主队-5%,  客队-8.5%
  >2500m:     主队-8%,  客队-13%

气温修正（开球时）：
  5-25°C: 0% | 25-30°C: -1% | 30-35°C: -3% | >35°C: -5% | <5°C: -3%

湿度修正：
  <70%: 0% | 70-85%: -1% | >85%: -2%

伤病修正（身价top3外场球员）：
  缺1个核心: -3% | 缺2个: -6% | 缺3个+: -8%(封顶) | 全员健康: 0%

主场观众修正：
  50%(中立): 0% | 60%: +1% | 70%: +2% | 80%+: +3%

λ_adj = λ × (1 + 海拔修正_主 + 气温修正 + 湿度修正 + 旅途修正 + 伤病修正 + 主场观众)
μ_adj = μ × (1 + 海拔修正_客 + 气温修正 + 湿度修正 + 旅途修正 + 伤病修正)
```

### 步骤9：泊松分布

```
P(主队进k球) = (λ^k × e^(-λ)) / k!
P(客队进j球) = (μ^j × e^(-μ)) / j!

计算k=0到6，j=0到6的所有组合（49个格子）
```

### 步骤10：Dixon-Coles τ修正（必须执行！）

```
ρ = -0.13

τ(0,0) = 1 - λμρ
τ(1,0) = 1 + μρ
τ(0,1) = 1 + λρ
τ(1,1) = 1 - ρ = 1.13
τ(其他) = 1

P_最终(比分x:y) = τ(x,y) × P(x|λ) × P(y|μ)
```

### 步骤11：汇总1X2概率

```
主胜 = 所有x>y的P(x,y)之和（x从1到6，y从0到x-1）
平局 = 所有x=y的P(x,y)之和（x从0到6）
客胜 = 所有x<y的P(x,y)之和（y从1到6，x从0到y-1）

总和应≈100%（截断误差<0.1%可忽略）
```

### 步骤12：竞彩去水

```
抽水率 = (1/胜赔率 + 1/平赔率 + 1/负赔率 - 1) × 100%

市场去水概率_胜 = (1/胜赔率) / (1/胜赔率 + 1/平赔率 + 1/负赔率)
市场去水概率_平 = (1/平赔率) / (同上)
市场去水概率_负 = (1/负赔率) / (同上)
```

### 步骤13：Edge Score

```
Edge_Score = (模型概率 - 市场去水概率) × 赔率

判定（竞彩基准）：
  ≥18%  = 🟢 买入
  10-18% = 🟡 边缘
  <10%  = 🔴 不买
```

### 步骤14：1/4 Kelly仓位

```
只对Edge≥10%的方向计算：

b = 赔率 - 1
p = 模型概率
q = 1 - p

f* = (1/4) × (b×p - q) / b

单场下注金额 = f* × 总本金
```
