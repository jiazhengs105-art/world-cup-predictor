# 2026世界杯预测模型

> 你不是在跟精算师对赌——你是在跟昨天看完揭幕战头脑发热的散户对赌。
> 这个模型帮你站在情绪的对面，用数据说话。

---

## 模型总览

```
                         ┌─ 定性情报采集层（新增 v3.0）─┐
                         │ 教练风格 · 战术克制 · 更衣室  │
                         │ 训练情报 · 裁判尺度 · 多语言  │
                         │ 信息可信度过滤 · 定性→量化   │
                         └────────────┬────────────────┘
                                      ↓  ±3pp/维度, ±5pp总计
最终概率 = Dixon-Coles泊松(
              环境修正(
                  行为金融修正(
                      多源强度指数(
                          真实Elo(55%) + 俱乐部质量(25%) + 近期状态(20%)
                      )
                  )
              )
          )

其中行为金融修正层捕获四种"数据来不及反应"的因素：
① 市场情绪溢出（抽水膨胀 → 量化）
② 标志性胜利余温（Elo滞后×时间衰减 → 量化）
③ 核心伤病影响力（身价占比×国家队xG参与率 → 量化）
④ 东道主/主场叙事压力（历史偏差数据库 → 量化）

定性情报采集层捕获"公开但非结构化"的信息（搜索引擎可达，但无数据库）：
⑤ 教练战术风格（压迫倾向×防守深度 → 比赛节奏因子）
⑥ 战术克制关系（边路对位×中场控制权 → 克制方向修正）
⑦ 球队内部氛围（队长更替×更衣室分裂信号 → 韧性修正）
⑧ 裁判尺度（场均黄牌×比赛中断倾向 → 进球时间分布修正）
```

## 第一章：真实 Elo 计算

### 1.1 获取原始 Elo

数据来源：World Football Elo Ratings (`eloratings.net`)

获取方式：WebFetch `https://www.eloratings.net/` 或使用 Kaggle 数据集 `International Football Elo Ratings (1872-2025)`

### 1.2 联合会偏差修正

Elo是联合会内零和博弈——对手池质量决定分数天花板。

| 联合会 | 修正值 | 原因 |
|--------|--------|------|
| **UEFA**（欧洲） | **0** | 55队深度竞争，基准线 |
| **CONMEBOL**（南美） | **−15** | 10队双循环无鱼腩，分数被结构性压低 |
| **CONCACAF**（中北美） | **−40** | 前6后29实力断崖，大量低风险刷分局 |
| **AFC**（亚洲） | **−25** | 日韩伊澳沙之外实力骤降 |
| **CAF**（非洲） | **−20** | 埃及塞内加尔之外竞争偏弱 |
| **OFC**（大洋洲） | **−55** | 新西兰独大，几乎全鱼腩 |

```
True_Elo = Raw_Elo + Confed_Adjustment
```

### 1.3 时间衰减

近期比赛权重更高，使用指数衰减：

```
Decay_Weight = e^(-ξ × days_since_match)
其中 ξ = 0.005（约半年后权重降至一半）
```

> 当需要精确到单场时，使用时间衰减；日常对比直接使用 True_Elo 即可。

## 第二章：多源强度指数

### 2.1 俱乐部质量分数

```
Club_Quality_Score = (Big5_Minutes_Pct × 0.5) + (TM_Value_Percentile × 0.5)
```

| 变量 | 含义 | 来源 |
|------|------|------|
| Big5_Minutes_Pct | 全队球员在五大联赛（英超/西甲/德甲/意甲/法甲）的出场时间占比 | FBref 球员页 |
| TM_Value_Percentile | 全队德转身价在48队中的百分位 | Transfermarkt 国家队页 |

### 2.2 近期状态

```
Recent_Form = PPG_last_10 × Opponent_Strength_Multiplier

其中：
- PPG_last_10 = 近10场正式比赛场均积分（胜=3, 平=1, 负=0）
- Opponent_Strength_Multiplier = 对手平均 Elo / 全球平均 Elo
```

近10场比赛优先取：世界杯预选赛 > 洲际杯赛正赛 > 友谊赛

### 2.3 合成最终强度

```
Final_Strength = True_Elo × 0.55 + Club_Quality_Score × 0.25 + Recent_Form × 0.20

其中 Club_Quality_Score 和 Recent_Form 需先归一化到 Elo 量纲（均值1500，标准差200）
```

## 第三章：Dixon-Coles 泊松模型

### 3.1 攻防系数

从最近20场国家队正式比赛提取：

```
联赛场均主场xG参考值 ≈ 1.50（国际赛事基准）
联赛场均客场xG参考值 ≈ 1.10（国际赛事基准）

主队进攻强度 = 主队近20场场均进球 / 全部48队场均进球
主队防守强度 = 主队近20场场均失球 / 全部48队场均失球
客队进攻强度 = 客队近20场场均进球 / 全部48队场均进球
客队防守强度 = 客队近20场场均失球 / 全部48队场均失球
```

> 如果有 xG 数据（欧洲预选赛、2022世界杯正赛），优先使用 xG；否则使用实际进球作为代理。

### 3.2 预期进球

```
主队λ = 基准主场xG(1.50) × 主队进攻强度 × 客队防守强度 × 主场优势因子
客队μ = 基准客场xG(1.10) × 客队进攻强度 × 主队防守强度

其中主场优势因子默认 = 1.15（约+15%主场xG加成），由环境模块进一步调整
```

Elo 差值也可以交叉校验 λ/μ 的合理性：

```
Elo差值 → 预期进球插值（经验公式）:
ΔElo=0   → λ≈1.35, μ≈1.15
ΔElo=50  → λ≈1.55, μ≈1.05
ΔElo=100 → λ≈1.75, μ≈0.95
ΔElo=200 → λ≈2.15, μ≈0.75
```

当纯攻防系数算出的 λ/μ 和 Elo 经验公式偏差 >20% 时，取加权平均（攻防系数×0.6 + Elo插值×0.4）。

### 3.3 泊松分布 → 比分概率

```
P(主队进k球) = POISSON(k, λ) = (λᵏ × e⁻λ) / k!
P(客队进j球) = POISSON(j, μ) = (μʲ × e⁻μ) / j!
```

### 3.4 Dixon-Coles τ修正

泊松假设两队进球独立，但实际 0-0、1-0、0-1、1-1 发生的频率更高。
τ 函数修正这四种比分：

```
τ(λ,μ,x,y) = 
  │ 1 − λμρ          若 x=0, y=0
  │ 1 + λρ            若 x=0, y=1
  │ 1 + μρ            若 x=1, y=0
  │ 1 − ρ             若 x=1, y=1
  │ 1                 其他

其中 ρ = −0.13（国际赛事经验值）
```

修正后的联合概率：

```
P(x,y) = τ(λ,μ,x,y) × Poisson(x|λ) × Poisson(y|μ)
```

### 3.5 汇总胜平负

```
主胜 = Σ(x>y) P(x,y)    （x从1到6+，y从0到x-1）
平局 = Σ(x=y) P(x,y)    （x=y从0到5）
客胜 = Σ(x<y) P(x,y)    （y从1到6+，x从0到y-1）
```

> 实际计算截断到双方各0-7球（8×8=64种比分组合），覆盖 >99.9% 的概率质量。

## 第四章：环境变量量化

所有环境修正**乘性叠加**到预期进球上：

```
λ_adjusted = λ_base × (1 + altitude_mod + temp_mod + humidity_mod + travel_mod + rest_mod + injury_mod)
μ_adjusted = μ_base × (1 + altitude_mod_away + temp_mod + humidity_mod + travel_mod_away + rest_mod + injury_mod_away)
```

### 4.1 海拔

| 海拔范围 | 修正系数 |
|---------|---------|
| 0-500m | 0% |
| 500-1500m | 主队 −2%, 客队 −3% |
| 1500-2500m | 主队 −5%, 客队 −8.5% |
| >2500m | 主队 −8%, 客队 −13% |

> 墨西哥城（阿兹特克体育场，2200m）是整个赛事中海拔最高的场地。

### 4.2 气温（开球时）

| 温度范围 | 修正系数 |
|---------|---------|
| 5°C – 25°C | 0%（理想区间） |
| 25°C – 30°C | −1% |
| 30°C – 35°C | −3% |
| >35°C | −5% |
| <5°C | −3% |

### 4.3 湿度

| 相对湿度 | 修正系数 |
|---------|---------|
| <70% | 0% |
| 70% – 85% | −1% |
| >85% | −2% |

### 4.4 旅途疲劳

| 赛前4天内飞行距离 | 修正系数 |
|-----------------|---------|
| <1500km | 0% |
| 1500-3000km | −2% |
| 3000-5000km | −4% |
| >5000km | −7% |

> 欧洲队跨大西洋 → 自动 −7%。南美队到北美 → −4%。美国本土东西海岸 → −4%。

### 4.5 时差

| 3天内穿越时区数 | 修正系数 |
|---------------|---------|
| 0-2 | 0% |
| 3-5 | −2% |
| >5 | −4% |

### 4.6 休整天数

| 距上一场比赛 | 修正系数 |
|------------|---------|
| <3天 | −5% |
| 3-4天 | −2% |
| 5+天 | 0% |

> 小组赛轮次间通常 3-4 天 → 默认 −2%。淘汰赛间隔更长。

### 4.7 伤病

| 缺阵情况 | 修正系数 |
|---------|---------|
| 全员健康 | 0% |
| 1名核心球员缺阵 | −3% |
| 2名核心球员缺阵 | −6% |
| 3名及以上核心缺阵 | −8%（封顶） |

> "核心球员"定义：全队德转身价排名前3的外场球员。

### 4.8 主场观众比例

| 主场观众% | 修正系数 |
|----------|---------|
| 50%（完全中立） | 0% |
| 60% | +1% |
| 70% | +2% |
| 80%+ | +3% |

> 美国在主场 vs 欧洲队 → 默认 +2%；加拿大多伦多/温哥华 → +2%。

## 第五章：行为金融修正

> 传统数据（Elo、身价、战绩）反映的是"已经发生的事"。
> 这一层捕捉的是"正在发生但数据还没来得及反应的事"——
> 散户情绪、叙事驱动、伤病突发、主场压力。
> 每一个"感觉"都有数学锚点。

### 5.1 市场情绪溢出（抽水膨胀法）

**机制**：重大事件（揭幕战东道主赢球、强队爆冷）通过可用性启发式影响散户，导致赔率被情绪性压缩。庄家通过提高抽水来对冲一边倒的订单流。

**量化方法**：直接读取抽水膨胀值。

```
正常抽水基准 = 11%（竞彩单关正常水平）
当前抽水 = (Σ隐含概率 − 1) × 100%
情绪溢价 = max(0, 当前抽水 − 11%)

修正映射：
| 情绪溢价 | 含义 | 热门队胜率 | 冷门队胜率 |
|---------|------|----------|----------|
| 0-3% | 正常波动 | 0 | 0 |
| 3-8% | 明显情绪 | −3pp | +3pp |
| 8-15% | 严重情绪 | −5pp | +5pp |
| >15% | 极度扭曲 | −8pp | +8pp |
```

**为什么 11%**：竞彩单关法定返奖率约 89%，即正常抽水 11%。这高于国际庄家的 5-7%，但低于竞彩串关的 27-29%。所有后续判断默认使用竞彩赔率。

> 实操：如果一场比赛抽水 < 14%（竞彩正常波动上限），情绪修正直接跳过。如果抽水飙到 18%+，自动应用 ±3pp 修正。

### 5.2 标志性胜利余温（Elo 滞后法）

**机制**：Elo 只看比分结果，不看比赛过程。一场"弱队淘汰强队"的标志性胜利对球队心理的提升，远超 Elo 更新所反映的。

**量化方法**：标志性胜利的"真实质量" − Elo 更新量，乘以时间衰减。

```
Step 1：识别标志性胜利
  条件：击败 Elo 比自己高 ≥ 150 的对手，且在正式比赛（非友谊赛）
  
Step 2：计算心理势能
  对手质量溢价 = (对手Elo − 本队Elo) / 400
  比赛性质权重 = 世界杯正赛 1.0 | 洲际杯 0.8 | 预选赛 0.6 | 友谊赛 0.3
  Elo实际变动 = 赛后Elo − 赛前Elo（按 Elo 公式的标准结果）
  
  心理势能 = 对手质量溢价 × 比赛性质权重 × 30

Step 3：时间衰减
  衰减因子 = e^(−0.02 × 距今场数)
  修正值 = 心理势能 × 衰减因子 / 100
  
  转换为胜率修正 → 封顶 ±5pp
```

**示例**：波黑（Elo ~1640）淘汰意大利（Elo ~1910）
```
对手质量溢价 = (1910 − 1640) / 400 = 0.675
比赛性质 = 预选赛附加赛 → 0.8（仅次于洲际杯）
心理势能 = 0.675 × 0.8 × 30 = 16.2
距今 2 场 → 衰减 = e^(−0.04) ≈ 0.96
修正值 = 16.2 × 0.96 / 100 ≈ +1.6pp（向上取整 +3pp 封顶）
```

**为什么 30 这个系数**：Elo 的 K 值约为 40（预选赛级别），一场爆冷大约交换 15-25 分。心理层面的额外溢价大约是 Elo 变动的 1.5-2 倍，体现在未来比赛的"韧性强于纸面实力"上。30 是经验校准值。

### 5.3 核心伤病影响力（身价占比法）

**机制**：4.7 节的伤病修正只区分"是否核心"，这章给出核心的准确定义和量化。

**量化方法**：

```
球员影响力 = (球员德转身价 / 全队总身价) × 0.5 
            + (球员国家队 xG参与率) × 0.5

其中：
- 身价占比 = 个人身价 ÷ 全队26人身价总和
- xG参与率 = (球员国家队 xG + 球员国家队 xA) ÷ 全队近10场总 xG

缺阵修正 = −影响力 × 降权系数(0.7) × 100
         → 转换为胜率 pp 修正，封顶 −8pp
```

| 球员影响力 | 分类 | 修正 | 示例 |
|----------|------|------|------|
| >15% | 头号核心 | −4pp | 戴维斯(加拿大)、姆巴佩(法国) |
| 10-15% | 二号核心 | −2pp | 戴维(加拿大) |
| 5-10% | 主力 | −1pp | |
| <5% | 角色球员 | 0 | |

> 多人缺阵时叠加但封顶 −8pp。如果头号核心缺阵但球队在同一位置有接近水平的替补（身价差距 <30%），修正减半。

### 5.4 东道主/主场叙事压力（历史偏差法）

**机制**：东道主首秀、关键战晋级、揭幕战——这些"叙事重量"超出正常主客场分析。历史数据有规律可循。

**量化方法**：查历史偏差数据库。

```
世界杯东道主首场表现偏差（1994-2026）：

| 年份 | 东道主 | 预期胜率(Elo) | 实际结果 | 偏差 |
|------|--------|------------|---------|------|
| 2022 | 卡塔尔 | ~20% | 0-2 负 | −0.3 球 |
| 2018 | 俄罗斯 | ~65% | 5-0 胜 | +3.2 球 |
| 2010 | 南非 | ~35% | 1-1 平 | +0.6 球 |
| 2006 | 德国 | ~70% | 4-2 胜 | +0.8 球 |
| 2002 | 韩国 | ~30% | 2-0 胜 | +1.5 球 |
| 1998 | 法国 | ~65% | 3-0 胜 | +0.5 球 |

东道主平均超预期：+0.9 球

但分层看：
| 东道主赛前 Elo | 首场方向 |
|---------------|---------|
| 强队（>1900） | 符合或略超预期 → +1pp |
| 中游（1700-1900） | 明显超预期 → +2pp |
| 弱队（<1700） | 方差极大 → 不加修正 |

关键区分：实际战力加成 vs 市场超额定价
- 中游东道主 → 历史真实加成 +1.5pp
- 中游东道主被散户情绪买过头 → 市场已定价 +5pp
- 净效果 = +1.5 − 5 = −3.5pp → 反而有利于对手！
```

**使用规则**：
1. 查表获取东道主的 Elo 档位
2. 获取"历史真实加成"（物理层面的主场作战能力提升）
3. 与 5.1 节的情绪溢出修正交叉校验——如果情绪溢价已经把东道主买过头了，历史真实加成可能被抵消甚至反转
4. 非东道主的普通主场球队 → 使用 4.8 节主场观众修正，不适用本节

### 5.5 行为金融修正叠加规则

四个因素**独立计算、线性叠加**，封顶 ±12pp：

```
行为修正总和 = 5.1情绪溢出 + 5.2标志胜利 + 5.3伤病 + 5.4叙事压力

总修正 = clamp(行为修正总和, −12pp, +12pp)
```

> 封顶理由：行为因素的本质是"市场来不及反应的信息"，不是"市场完全瞎了"。超过 12pp 的修正意味着你在赌市场犯了系统性错误——这在成熟博彩市场极少发生。

### 5.6 应用流程

将行为金融修正注入主模型：

```
原始概率（第三、四章输出）
    ↓
行为金融修正层（第五章）：
  P_adjusted(结果) = P_raw(结果) + Σ(行为修正值)
  然后重新归一化使 ΣP = 100%
    ↓
最终概率
    ↓
Edge Score 计算（第 X 章）
```

## 第六章：数据采集流程

当被调用预测一场比赛时，按以下顺序获取数据。**结构化数据（Step 1-5）为必选，定性情报（Step 6-9）为必选——这是 v3.0 与之前版本的核心区别。**

### 第一轮：结构化数据采集

#### Step 1：Elo 评分
```
WebFetch https://www.eloratings.net/
→ 提取两队当前 Elo
→ 应用联合会修正 → True_Elo
```

#### Step 2：德转身价 + 俱乐部质量
```
WebSearch "{Team Name} national team squad market value 2026 transfermarkt"
→ 获取全队总身价 + 每位球员所属俱乐部
→ 计算 Big5_Minutes_Pct + TM_Value_Percentile
```

#### Step 3：近期战绩
```
WebSearch "{Team Name} recent results 2025 2026 last 10 matches"
→ 提取近10场正式比赛结果
→ 列出每场对手及近似 Elo
→ 计算 PPG_last_10 × Opponent_Strength_Multiplier
```

## 第七章：定性情报采集与量化映射

> 这是 v3.0 的核心新增。数学模型和资深赌球专家的差距不在结构化数据（大家都能搜到 Elo），
> 而在对"公开但非结构化"信息的系统性采集和量化转化能力。
>
> 本章解决两个问题：
> 1. 搜什么？（第六章 Step 6-10 已定义搜索协议）
> 2. 搜到之后怎么转化成概率修正？（本章定义映射框架）

### 7.1 设计原则

```
定性情报 → 量化修正 的三个约束：

1. 每维度修正不超过 ±3pp
2. 所有定性维度叠加封顶 ±5pp
3. 修正方向必须基于"该信息是否已被市场定价"来判断
   - 已被市场定价 → 不修正（市场不傻）
   - 未被市场定价 → 修正（你找到了信息差）
```

> 为什么封顶 ±5pp？定性情报的本质是"搜索引擎能找到的公开信息"——
> 如果搜索引擎能找到，精算师也能找到（虽然他们可能没找）。
> ±5pp 意味着定性情报最多贡献约 15-25% 的 Edge Score 来源，
> 主要 Alpha 仍来自行为金融修正和 Dixon-Coles τ 修正。

### 7.2 教练战术风格 → 比赛节奏与进球期望修正

从 Step 6 搜索提取的信息进行量化：

**进攻节奏分类**：

| 节奏类型 | 特征 | λ 修正（双方） |
|---------|------|---------------|
| 快节奏（高位压迫+快速转换） | 球权交换频繁、射门总数高 | 双方 λ 各 +3%（比赛更开放） |
| 中节奏（控球+选择性压迫） | 有控制的推进、不盲目射门 | 0%（基准） |
| 慢节奏（低位防守+拖时间） | 进攻投入少、追求低比分 | 双方 λ 各 −3%（比赛更沉闷） |

**防守深度分类**：

| 防守策略 | 特征 | 平局概率修正 |
|---------|------|------------|
| 双高位（两队都高位防守） | 大量对攻、进球多 | 平局 −2pp（分出胜负概率高） |
| 一高一低 | 差距明显 | 平局 +1pp（强队可能在 60 分钟后陷入僵局） |
| 双低位（两队都保守） | 极少冒险、等待对方犯错 | 平局 +2pp（0-0 和 1-1 概率结构性上升） |

**定位球依赖度**：

| 依赖程度 | 特征 | 修正 |
|---------|------|------|
| 高度依赖（球队 30%+ 进球来自定位球） | 进攻创造力不足，靠定位球破局 | 平局 +1pp（运动战进球困难） |
| 一般 | 定位球进球占比 15-25% | 0 |
| 不依赖 | 定位球占比 <15% | 0 |

### 7.3 战术克制关系 → 克制方向修正

从 Step 7 搜索提取"谁克制谁"：

```
克制判定矩阵：

A 的进攻强侧（左路）→ 面对 → B 的防守弱侧（右路，边卫进攻前插留下空档）
→ A 有明确突破口 → A 胜率 +2pp，修正从 B 胜率扣除

A 的中场创造力（依赖单一核心）→ 面对 → B 的针对性绞杀战术
→ B 的战术针对性优于 A → B 胜率 +2pp，修正从 A 胜率扣除

A 的高位防线 → 面对 → B 的快速反击（姆巴佩式）
→ B 的反击克制 A 的高位 → B 胜率 +2pp

二者无明显克制关系 → 0pp
```

**判断标准**：需要至少 2 个独立战术分析来源指向同一方向，才触发克制修正。
单一来源的分析不触发（防止某个分析师的主观偏见）。

### 7.4 球队内部氛围 → 韧性修正

从 Step 8 搜索提取：

| 氛围信号 | 证据标准 | 修正 |
|---------|---------|------|
| 🟢 积极团结 | 教练/球员赛前表态一致，无负面报道，队长稳定 | 0（基准） |
| 🟡 轻微动荡 | 队长刚更换且新队长未经大赛考验；赛前关键球员续约/转会传闻 | 大比分落后时崩盘概率 ↑ → 对手胜率 +1pp |
| 🔴 公开分裂 | 跟队记者报道更衣室矛盾、教练权威受质疑 | 球队 xG −3%（场上执行打折） |

⚠️ **触发标准极高**：🐍 信号仅在 2 个以上独立来源交叉验证时才触发。
单一来源的"矛盾"报道 → 视为 C 级信度 → 修正减半（即 −0.5pp）。

### 7.5 裁判尺度 → 进球时间与比赛中断修正

从 Step 9 搜索提取：

| 裁判风格 | 场均黄牌 | 修正 |
|---------|---------|------|
| 宽松尺度 | <3.0 张/场 | 比赛流畅 → 实际比赛时间 ↑ → 双方 λ 各 +2% |
| 正常尺度 | 3.0-4.5 张/场 | 0（基准） |
| 严格尺度 | >4.5 张/场 | 比赛频繁中断 → 实际比赛时间 ↓ + 红牌风险 ↑ → 双方 λ 各 −2%，平局 +1pp |

**红牌倾向**：如果该裁判近 10 场执法中红牌率 >15%（即至少 2 张红牌），
额外修正：任意一方少打一人概率 +5% → 在概率汇总时微调比分矩阵的尾部。

### 7.6 赛前训练情报 → 首发/阵型变化预期

从 Step 10 母语搜索提取：

```
赛前 24-48h 训练情报的三种信号：

信号 1：关键球员恢复训练 → 伤病修正从未知恢复为确定
  例："三笘薰昨日已参加全队合练" → 5.3 节伤病修正从 −2pp 撤销为 0

信号 2：训练中尝试新阵型 → 战术不确定性上升
  例："森保一在封闭训练中尝试了 4-3-3" → 赛前预期被打乱 → 模型方差 ↑
  → 不直接修正概率，但降低本场预测置信度，Kelly 仓位自动收缩

信号 3：多名替补球员穿主力背心 → 首发可能轮换
  例："训练中町野修斗穿主力背心" → 可能替代某核心首发
  → 如果被替代者是核心（影响力>10%）→ 按 5.3 公式计算伤病修正的 50%（因为未确认）
```

### 7.7 定性情报叠加规则

```
所有定性维度的修正，线性叠加到行为金融修正后的概率上：

P_pre_final = P_after_ch5 + Σ(定性修正值)
P_final = normalize(P_pre_final)  # 使 ΣP = 100%

定性修正总上限 = clamp(定性修正总和, −5pp, +5pp)
```

**与行为金融修正的关系**：
- 行为金融修正（第五章）和定性情报修正（第七章）**独立计算、各自封顶**
- 行为金融修正封顶 ±12pp，定性情报修正封顶 ±5pp
- 两者叠加后的总修正封顶 **±15pp**（行为金融仍设 ±12pp 硬顶，剩余 ±3pp 留给定性）
- 实际框架：`clamp(行为修正, −12pp, +12pp) + clamp(定性修正, −3pp, +3pp)` → 总封顶 ±15pp

> 定性修正封顶比第五章（±12pp）更窄，因为其信噪比更低。
> 搜索引擎能找到的情报，精算师也能找到；真正的信息壁垒在数据不在搜索引擎里的东西。

## 第八章：分数凯利仓位控制

> 概率告诉你该买什么，凯利告诉你该买多少。两者是独立的决策系统。
> 40% 概率正确但仓位过大 → 连续落空三次本金腰斩 → 模型正确、策略破产。

### 8.1 全凯利 vs 分数凯利

**全凯利公式（体育博彩严禁使用）**：

```
f* = (bp − q) / b

其中：
- b = 十进制赔率 − 1（净赔率）
- p = 模型真实概率
- q = 1 − p
- f* = 建议投入本金比例
```

全凯利的问题：
- 假设模型概率是完美精确的 → 实际不可能
- 假设你可以无限次重复下注 → 世界杯只有 64 场
- 方差极大 → 连续 3 场不中（在 40% 概率下发生率为 21.6%），本金亏损 60%+

**1/4 分数凯利（本模型强制使用）**：

```
f* = (1/4) × (bp − q) / b
```

| 模型概率 | 赔率 | 全凯利仓位 | 1/4 Kelly 仓位 |
|---------|------|----------|---------------|
| 40% | 3.43 | 16.3% | **4.1%** |
| 35% | 3.25 | 10.8% | **2.7%** |
| 55% | 1.71 | 18.4% | **4.6%** |
| 50% | 2.00 | 25.0% | **6.3%** |

### 8.2 决策规则

```
IF Edge_Score < 10%（🔴 负/微利）
  → f* = 0.00%（空仓，无论凯利公式计算结果如何）

IF Edge_Score 10-18%（🟡 边缘）
  → 使用 1/8 Kelly（半仓）：f* = (1/8) × (bp − q) / b
  → 理由：边际机会，降低方差保护本金

IF Edge_Score ≥ 18%（🟢 正期望）
  → 使用 1/4 Kelly：f* = (1/4) × (bp − q) / b

IF 单注 Kelly 仓位 < 1%
  → 向上取整到 1%（金额太小不具备操作价值）

IF 单注 Kelly 仓位 > 10%
  → 截断到 10%（单场世界杯比赛的残差太大，不接受 >10% 本金暴露）
```

### 8.3 二串一套用

```
串关真实胜率 = P(A结果) × P(B结果)
串关赔率 = Odds_A × Odds_B
串关净赔率 b_parlay = 赔率_串关 − 1

凯利计算：
  p = 真实坍缩胜率
  b = 串关净赔率
  f* = (1/4) × (b × p − (1−p)) / b

复合抽水率 = 1 − 1/(1/odds_A1 + 1/odds_A2 + 1/odds_A3) × 1/(1/odds_B1 + 1/odds_B2 + 1/odds_B3)
            × 1/(1/odds_parlay_implied)
  （简化：复合抽水 ≈ 1 − (1−抽水A)×(1−抽水B)）
  
⚠️ 复合抽水 > 25% → 串关自动否决（抽水吃掉太多期望）
```

### 8.4 残差警告（每注必输出）

在给出 Kelly 仓位时，必须标注一个"模型盲区"：

```
残差来源清单（每次选最相关的 1 条标注）：

□ 裁判尺度极端偏向一方（主场哨/赛后舆论压力）
□ 突发的赛前 2 小时内伤病/首发变更（模型来不及反应）
□ 极端天气（暴雨/暴雪导致比赛变成定位球大战，λ 模型失效）
□ 红牌改变比赛结构（10v11 的泊松分布完全不同于 11v11）
□ 某队"战略性放弃"小组赛（已出线/已淘汰 → 全替补 → 身价数据失效）
□ 点球大战心理（淘汰赛专用——泊松模型完全不考虑点球）
```

## 第九章：Edge Score 统一决策系统

### 8.1 公式

```
Edge_Score = (P_model − P_market) × Odds

其中：
- P_model = 模型最终概率（经过第一至五章全部修正后）
- P_market = 市场去水概率（博彩公司隐含概率 ÷ 总隐含概率）
- Odds = 该结果的十进制赔率
- Edge_Score = 每下注 1 单位，数学期望净利润（%）
```

### 8.2 决策规则（竞彩基准）

| Edge Score | 判定 | 动作 |
|-----------|------|------|
| **≥ 18%** | 🟢 正期望 | **买入。** 竞彩11%抽水 + 模型误差都打穿了 |
| **10% – 18%** | 🟡 边缘 | 可买可不买。取决于定性判断把握 |
| **< 10%** | 🔴 负/微利 | **不买。** 竞彩抽水一刮就没了 |

### 8.3 "买强队"和"买弱队"是否应该不同

不需要。公式自动处理——低赔率天然压制 Edge。竞彩更高的抽水进一步拉高了门槛。

| 方向 | 赔率范围 | 到 18% 需概率差 |
|------|---------|------------------|
| 强队（大热门） | 1.20 – 1.60 | 需 11-15pp 差距 |
| 中游 | 1.80 – 2.50 | 需 7-10pp 差距 |
| 弱队（冷门） | 3.00 – 5.50 | 需 3-6pp 差距 |

> 在竞彩上买强队极难——抽水 11% + 低赔率双重压制。中游和冷门才是主战场。

### 8.4 为什么阈值是这三个数（竞彩版）

- **18%**：竞彩抽水 11% + 模型残差 ~5pp + 安全垫 2pp
- **10%**：竞彩抽水 11% + 模型残差 ~3pp 的保底线
- **低于 10%**：扣掉 11% 抽水后几乎确定是负期望

### 8.5 竞彩 vs 国际庄家

| | 国际庄家（Bet365等） | 竞彩（中国体彩） |
|------|------|------|
| 正常抽水 | 5-7% | **11%** |
| 情绪基准 | 7% | **11%** |
| 🟢 阈值 | 15% | **18%** |
| 🟡 阈值 | 8% | **10%** |
| 对手质量 | 散户 + 部分职业 | **几乎全是散户**——信息效率更低 |
| 合法性 | ❌ 国内不合法 | ✅ |

> 竞彩抽水更高，但对手的信息效率更差。你的 alpha 来源不变——找散户情绪集体犯错的时候。只是入场费贵了，需要更大的概率优势才能出手。

### 8.6 实战速查（竞彩版）

不必每次都手动算。看三件事：

```
1. 抽水 > 14%？ → 竞彩情绪在一边倒（超正常11%基准）
2. 模型的"冷门方向"概率比市场高 > 5pp？ → 你和散户站在对面
3. Edge > 18%？ → 出手

三个都是 Yes → 🟢
缺一个 → 🟡 或 🔴
```

## 局限与使用须知

1. **xG数据稀疏**：非欧洲/南美预选赛基本没有公开xG，使用实际进球作代理——信息量打折扣
2. **俱乐部数据盲区**：只覆盖五大联赛球员，在荷甲/葡超/巴甲/MLS效力的球员被系统性低估
3. **伤病延迟**：Transfermarkt伤病更新时间不确定，可能遗漏赛前24小时的变化
4. **天气不确定性**：比赛日14天前的天气预报误差大，越接近比赛越准
5. **小组赛策略差异**：48队扩军后小组第三也可能出线，弱队死守动力增强 → 平局概率结构性上升
6. **定性情报信噪比**（v3.0）：搜索引擎返回的教练发言可能是烟雾弹（没人会在发布会上说真话），更衣室矛盾的报道可能是记者为流量编的，多语言搜索的机器翻译可能误读语气。定性修正封顶 ±3pp 本身就是对这一信噪比的承认
7. **这不是水晶球**：模型提供的是系统性概率基线，一场比赛的随机性远大于任何模型能捕捉的。长期正EV需要大量样本兑现
